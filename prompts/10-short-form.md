# 10. 短尺SNS向け編集 (Short-form 9:16)

YouTube Shorts / Instagram Reels / TikTok 向けの最適化編集。9:16リサイズ、冒頭フック、字幕、CTAなど。

---

## 1. 16:9 → 9:16 リサイズ（被写体トラッキング）

**ユースケース**: 横動画をショート動画に転用
**入力**: 16:9の横動画

**プロンプト**:
```
この16:9の横動画を、被写体を中心に9:16の縦動画にリサイズしてください。
- メインの被写体（人物）を画面中央に常に保持
- カメラパンや被写体移動に追従して縦フレームを動的に調整
- 上下の見切れる部分は被写体を最優先で保持
- 必要に応じて上下にアウトペイントで背景を補完
- フレーム間でクロップ位置が滑らかに動く（カクつきなし）
- 元の音声を完全保持
```

**コツ**: 「被写体追従」を指定しないと棒立ち固定クロップになる。
**ネガティブ**: 被写体の見切れ、クロップのジッター、急なフレーム移動

---

## 2. 冒頭3秒フック生成（テロップ + ズーム）

**ユースケース**: SNSでスクロールを止めるフック
**入力**: 通常の動画

**プロンプト**:
```
この動画の冒頭3秒に、視聴者がスクロールを止めるフックを追加してください。
- 0秒〜0.5秒: 動画の最もインパクトのある瞬間を一瞬挿入
- 0.5秒〜2秒: 大きな赤色テロップ（「○○な方法、知ってる？」など内容に応じた問いかけ）
- 2秒〜3秒: 軽いズームインと音響効果（スウィッシュ）
- テロップは画面中央に2行以内
- フォントは太く視認性の高いもの（Noto Sans Bold等）
- 3秒以降は元の動画を再生
```

**コツ**: 「最もインパクトのある瞬間を一瞬挿入」がClick-through率を上げる。
**ネガティブ**: 平凡な冒頭、テロップの読みにくさ、テンポの遅さ

---

## 3. テンポ良いジャンプカット編集

**ユースケース**: トーク動画のテンポ改善
**入力**: 一人語りの動画

**プロンプト**:
```
この動画を、テンポの良いジャンプカット編集に再構成してください。
- 沈黙・「えーと」「あのー」などのフィラーをすべてカット
- 文と文の間の不要な間を0.2秒以下に圧縮
- 視覚的にも変化のあるジャンプカット（背景の微妙な変化）
- カット間に視聴者が違和感を感じない自然なリズム
- 元の発話内容は完全に保持
- カット率: 1秒あたり1〜2カットを目安
```

**コツ**: 「フィラーをすべてカット」だけで動画は3〜4割短くなる。
**ネガティブ**: 発話の途中で切れる、不自然な早口感、繋ぎ目の違和感

---

## 4. 字幕焼き付け（カラオケ風）

**ユースケース**: SNS視聴者の無音再生対応
**入力**: 音声付き動画

**プロンプト**:
```
この動画の発話内容を、カラオケ風の字幕として焼き付けてください。
- 発話のタイミングに正確にシンク
- 1行あたり最大15文字、最大2行
- フォントは太く視認性の高いゴシック（白文字 + 黒縁取り）
- 現在発話中の単語のみ黄色にハイライト
- 字幕は画面下部、安全エリア内に配置
- 文節ごとに区切って表示
- 句読点で改行
```

**コツ**: 「現在の単語をハイライト」がカラオケ感の核。
**ネガティブ**: 字幕の遅延、見切れ、長すぎる行

---

## 5. エンディングCTA（フォロー&いいね）

**ユースケース**: SNSのコンバージョン強化
**入力**: 通常の動画

**プロンプト**:
```
この動画の末尾3秒に、エンゲージメント促進のCTAを追加してください。
- 「フォロー」「いいね」「コメント」のアイコンを画面下部に配置
- アイコンは順に出現（0.5秒ごと）
- 各アイコンに軽い跳ねるアニメーション
- 「もっと見たい人はフォロー！」のテロップを画面上部に
- 背景は元の動画末尾フレームを保持（オーバーレイのみ追加）
- 効果音（ポップ音）をアイコン出現時に
- 全体は控えめで邪魔にならない
```

**コツ**: 「跳ねるアニメ」「ポップ音」でクリックを促す。
**ネガティブ**: 過剰な装飾、視認性低下、画面の埋め尽くし

## 6. リール用ループ加工

**ユースケース**: Instagram Reelsの再生回数稼ぎ
**入力**: 短い動画クリップ

**プロンプト**:
```
この動画をシームレスにループするReels用クリップに加工してください。
- 末尾フレームから冒頭フレームへ滑らかに繋がる補間フレームを末尾に追加
- 視聴者がループに気づかない自然さ
- 全体の尺を15秒以内に調整
- 9:16アスペクト比に最適化
- 音声もループ点でクロスフェード
- 冒頭3秒に最も視覚的に強い瞬間を配置
```

**コツ**: 「ループに気づかれない」と書くと境界処理が丁寧になる。
**ネガティブ**: 巻き戻し感、ループ点の段差、テンポの悪さ

---

## 7. TikTok風ステッカー・絵文字追加

**ユースケース**: TikTokネイティブ感の演出
**入力**: 通常の動画

**プロンプト**:
```
この動画に、TikTok風のステッカー・絵文字オーバーレイを追加してください。
- 動画の重要な瞬間に絵文字を出現させる（🔥😂💯など内容に応じて）
- 絵文字は0.5秒ほどで現れて消える
- サイズは画面の10〜15%
- 出現時に軽いスケールアニメーション（バウンス）
- 配置は被写体を遮らない位置
- 全体で4〜6個程度（過剰でない）
- 効果音（ポンッ）を出現時に
```

**コツ**: 「過剰でない・遮らない」が品位を保つ鍵。
**ネガティブ**: 絵文字の埋め尽くし、被写体への重ね、低品質な配置

## 8. 倍速 + テンポ調整（Reels向け）

**ユースケース**: 冗長な動画をReels化
**入力**: 30〜60秒の動画

**プロンプト**:
```
この動画をReels向けに最適化（テンポ調整）してください。
- 全体を1.2倍速に
- 沈黙・無音部分は0.5秒以下に圧縮
- カットのリズムを1〜2秒間隔に
- 音声はピッチ保持で倍速対応
- BGMを軽快なテンポのものに置換可
- 全体の尺を15〜30秒に収める
- 冒頭3秒にフックとなる瞬間を配置
```

**コツ**: 「ピッチ保持」「無音圧縮」のセットでテンポが激変。
**ネガティブ**: 早口すぎる、音声の歪み、繋ぎ目の違和感

---

## 9. 3秒ごとのシーン切替リズム化

**ユースケース**: TikTok的なリズム編集
**入力**: 複数シーンの動画

**プロンプト**:
```
この動画を、3秒ごとのシーン切替を持つリズム編集に再構成してください。
- 3秒ごとに視覚的に異なるシーンに切替
- 切替はジャンプカットまたは軽いトランジション
- BGMのビートに切替タイミングを合わせる
- 各シーンには最低1つの動きや変化
- 全体の尺は15〜30秒
- 9:16アスペクト比に最適化
- 視聴者の注意が継続するテンポ
```

**コツ**: 「ビートに切替を合わせる」でプロ感が出る。
**ネガティブ**: 単調な切替、ビートズレ、退屈な静止

---

## 10. ハイライト抽出（30秒 → 9秒の要約）

**ユースケース**: 長尺動画のSNS用ダイジェスト
**入力**: 30秒〜数分の動画

**プロンプト**:
```
この動画から、SNS用のハイライト9秒を抽出してください。
- ハイライト判定基準: 動きが大きい瞬間、表情の変化、決定的場面
- 抽出した瞬間を3秒×3カット（または2秒×4〜5カット）で構成
- カット間は軽いジャンプカット
- 冒頭3秒に最強の引きを配置
- BGMまたは元音声を背景に
- 全体のテンポはリズミカル
- 9:16アスペクト比に最適化
```

**コツ**: 「冒頭3秒に最強の引き」はSNS編集の鉄則。
**ネガティブ**: 退屈な開始、時系列順、ハイライト不足
