# 05. オブジェクト編集 (Object Add / Remove / Replace)

映像内のオブジェクトを追加・削除・置換する。背景はそのままに、特定の物体だけを編集する。

---

## 1. 映像内の不要な人物を削除

**ユースケース**: 通行人が映り込んだ素材のクリーンアップ
**入力**: メイン被写体の背後に通行人がいる映像

**プロンプト**:
```
この動画の背景にいる通行人（メインの被写体以外の人物）をすべて自然に削除してください。
- メインの被写体は完全に保持
- 削除部分を背景（道路・建物・床）で違和感なく埋める
- カメラパンや被写体の動きに追従して整合的に処理
- 影と光の状態も背景に応じて再構成
- 元の音声から人物関連の雑音を軽減
- フレーム間でちらつかないよう連続性を確保
```

**コツ**: 「フレーム間の連続性」を入れないと削除部分がチラつく。
**ネガティブ**: 人型のシルエット残存、繰り返しパターン、不自然な穴

---

## 2. 看板やロゴの文字を別ブランドに差し替え

**ユースケース**: 版権配慮、商品名差し替え
**入力**: 看板やロゴが映っている映像

**プロンプト**:
```
この動画に映っている看板の文字を、元のフォント・色・配置を維持したまま「[新ブランド名]」に差し替えてください。
- フォントの書体・太さ・色を完全に維持
- 看板の素材感・反射・経年変化も維持
- カメラパンや遠近変化に追従
- フレーム間で文字が安定（揺れ・歪みなし）
- 周辺要素（建物・看板の枠）は変更しない
```

**コツ**: 文字差し替えは特にフレーム間安定性が重要。
**ネガティブ**: 文字の歪み、ちらつき、周辺要素の意図しない変化

---

## 3. 持ち物（コップ → マグカップ）を置換

**ユースケース**: 商品紹介、PR動画
**入力**: 人物がコップを持って話している映像

**プロンプト**:
```
この動画で被写体が手に持っているコップを、白い陶器のマグカップに置き換えてください。
- 手の持ち方・指の位置は完全に保持
- マグカップは取っ手付きの一般的な形状
- マグカップへの光と影は元のシーンと整合
- 中身の液体（コーヒー想定）は保持
- カメラの動きに追従
- 持ち上げる動作や口元への移動でも整合性を維持
```

**コツ**: 手の持ち方を保持しつつ取っ手付きに変えるのは難易度高。光の整合を最優先。
**ネガティブ**: 手の握り方の不自然さ、マグカップの浮き、液体の漏れ

---

## 4. 映像内に物体（猫）を追加

**ユースケース**: シーンに被写体を後付け
**入力**: 室内のソファが映る映像

**プロンプト**:
```
この動画のソファの上に、自然に座っている茶トラの猫を追加してください。
- 猫はカメラに対してやや横向きに座る
- 大きさは実寸（成猫サイズ）
- 元のソファのテクスチャと整合し、座面が微妙にへこむ
- ライティングは元のシーンに完全整合
- 影をソファに正しく投影
- 軽い尻尾の動きと耳の微動を持たせる（自然な生命感）
```

**コツ**: 「微妙にへこむ」「軽い動き」で違和感を消す。
**ネガティブ**: 浮いた猫、不整合な影、過度な動き

---

## 5. 自動車のメーカー・色を変更

**ユースケース**: 車種変更、版権配慮
**入力**: 駐車中または走行中の自動車

**プロンプト**:
```
この動画に映っている自動車を、メーカーと色を変更してください。
- 元: シルバーのセダン
- 変更後: 黒のスポーツクーペ
- 車体の位置・サイズ・パースは元と一致
- カメラパンや走行に追従
- ライティング・反射・影は元のシーンに整合
- ナンバープレートはぼかし処理
- 周辺要素（人物・背景）は変更しない
```

**コツ**: 元と変更後の車種を具体的に指定。
**ネガティブ**: 車体の歪み、ナンバープレートの読める残存、影の不一致

---

## 6. 食卓に新しい料理を追加

**ユースケース**: フードコンテンツの素材追加
**入力**: 食卓に料理が並んでいる映像

**プロンプト**:
```
この食卓の空きスペースに、湯気の立つ味噌汁のお椀を追加してください。
- 既存の料理・食器は完全に保持
- 味噌汁は和食器（黒い漆椀）に入った状態
- 湯気は微妙に立ち上る（過剰でない）
- ライティングと影は食卓全体と整合
- カメラの動きに追従
- 料理の配置として自然な位置
```

**コツ**: 湯気を「過剰でない」と指定して写実感を保つ。
**ネガティブ**: 湯気の過剰演出、食器の浮き、影の不整合

---

## 7. 壁の絵画・ポスターを差し替え

**ユースケース**: インテリア紹介、ブランド配慮
**入力**: 壁にポスターが貼られた室内映像

**プロンプト**:
```
この動画の壁に貼られているポスターを、別の指定画像に差し替えてください。
- ポスターの位置・サイズ・パースは元と一致
- 紙の質感・反射・微妙なシワは保持
- ライティングと影は壁面と整合
- カメラパンに追従
- フレーム間で画像が安定
- 周辺の壁・家具は変更しない
```

**コツ**: 紙の質感・反射の保持が写真合成感を解消する鍵。
**ネガティブ**: ポスターの浮き、貼り絵感、画像の揺れ

---

## 8. 飛行物体（鳥・ドローン）を追加

**ユースケース**: 空の映像にアクセント追加
**入力**: 空が映る屋外映像

**プロンプト**:
```
この動画の空に、滑空する鳥の群れ（カモメ約5羽）を追加してください。
- 鳥は画面の中〜遠距離を横切るように飛行
- 飛行は自然な編隊（V字 or 散開）
- 鳥のサイズは遠近に応じて適切
- ライティングは空の色温度に整合
- 鳥の影は地面に投影しない（高高度想定）
- 元の音声に控えめな鳥の鳴き声を追加
```

**コツ**: 「中〜遠距離」「群れの編隊」など具体指定で違和感を消す。
**ネガティブ**: 鳥の浮き、過剰な数、影の不整合

---

## 9. 床のオブジェクト（家具）を整理して削除

**ユースケース**: ミニマル空間の演出、ビフォーアフター
**入力**: 物が散らかった部屋の映像

**プロンプト**:
```
この動画の床にある散らかった物（衣類、書類、雑貨）をすべて削除し、床面を空ける整理後の状態にしてください。
- 大型家具（ソファ・テーブル）は保持
- 床に散らかった小物のみ削除
- 削除部分の床面（木目・カーペット等）を整合的に再構成
- 影と光の状態も再計算
- カメラの動きに追従
- フレーム間で安定
```

**コツ**: 「大型家具は保持・小物のみ削除」で意図を明確化。
**ネガティブ**: 大型家具まで消える、床の不自然な穴、繰り返しパターン

---

## 10. 服のロゴ・ブランドマークを除去

**ユースケース**: 版権配慮、汎用映像化
**入力**: ブランドロゴ入りの衣服を着た人物

**プロンプト**:
```
この動画で被写体の衣服に付いているブランドロゴを、自然に除去してください。
- 衣服のテクスチャ・色・シワは完全に保持
- ロゴ部分を周辺の生地パターンで違和感なく埋める
- カメラの動きや被写体の動作に追従
- ロゴへの光と影の状態を周辺と整合
- フレーム間でちらつかない
- 衣服のシルエットは変えない
```

**コツ**: 「周辺の生地パターンで埋める」と明示するときれいに消える。
**ネガティブ**: ロゴ残存、衣服の不自然な凹み、ちらつき
