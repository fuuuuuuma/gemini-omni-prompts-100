# 04. 背景・環境置き換え (Background Replacement)

被写体を保持したまま、背景・環境を別のロケーションや時間帯・天候に置き換える。グリーンバック不要のセグメンテーション置換。

---

## 1. 屋内 → 屋外（カフェ → ビーチ）

**ユースケース**: 自宅収録のVlogを旅先風に
**入力**: 室内で人物が話している映像

**プロンプト**:
```
この動画の背景を、カフェ室内から南国のビーチに置き換えてください。
- 被写体（人物・衣服・髪・手元の小物）は完全に保持
- 背景は青い海、白い砂浜、ヤシの木、明るい日差し
- 被写体のライティングを屋外の自然光に整合させる
- 髪や衣服に海風によるそよぎを微妙に追加
- 影の方向と硬さを屋外日中に再計算
- 元の音声に環境音（波音、軽い風）を控えめに重ねる
```

**コツ**: 「ライティングを整合」を入れないと被写体だけ室内光のまま浮く。
**ネガティブ**: 被写体の輪郭の不自然な縁、影の不一致、室内反射光の残存

---

## 2. 日常背景 → ファンタジー世界

**ユースケース**: 物語コンテンツの導入
**入力**: 街中で人物が立っている映像

**プロンプト**:
```
この動画の背景を、現代の街並みからファンタジー世界の中世風城下町に置き換えてください。
- 被写体（人物のみ）は完全に保持
- 背景は石畳、木造の家屋、霧、たいまつの炎
- 空はやや曇り、淡い夕焼け色
- 被写体のライティングを背景に整合
- 全体の色調を温かみのあるオレンジ寄りに統一
- 元の音声に環境音（風、遠くの市場の喧騒）を加える
```

**コツ**: 「人物のみ保持・衣服は背景に合わせて補正」も指定可。
**ネガティブ**: 現代要素の残存（看板、車）、被写体の浮き、光の不一致

---

## 3. グリーンバック不要の即時背景置換

**ユースケース**: クロマキー撮影なしで背景差し替え
**入力**: 通常背景で撮影した人物トーク映像

**プロンプト**:
```
この動画の背景を、被写体を保持したまま指定の背景に差し替えてください。
- 被写体の輪郭をフレームごとに正確にセグメンテーション
- 髪の毛の細かい束、半透明部分も自然に処理
- 新しい背景: 落ち着いたブックライブラリ（暖色の間接照明、本棚）
- 被写体のライティングを新背景に整合
- 影は床面と背景に自然に投影
- 元の音声と被写体の動きを完全保持
```

**コツ**: 「髪の毛の細かい束」を明示するとセグメンテーション品質が上がる。
**ネガティブ**: ハロー（後光）、髪の輪郭の欠け、フリッカー

---

## 4. 同じ部屋を別の時代（昭和レトロ）に変更

**ユースケース**: 時代設定の表現
**入力**: 現代の和室

**プロンプト**:
```
この動画の和室を、1970年代昭和レトロの茶の間に置き換えてください。
- 被写体（人物）は完全に保持
- 部屋の家具をブラウン管テレビ、卓袱台、座布団、レトロな扇風機に置換
- 壁の色を黄ばんだ漆喰調に
- 天井の照明を裸電球の黄色い光に
- カーテンや畳の色を当時の質感に
- 全体に時代特有の埃っぽさと退色を表現
```

**コツ**: 「ブラウン管テレビ」など具体的アイテム指定で時代感が出る。
**ネガティブ**: 現代家電の残存、過度な退色、被写体への色被り

---

## 5. 都市 → 宇宙ステーション内部

**ユースケース**: SFコンテンツ、テック系演出
**入力**: 街中での歩行映像

**プロンプト**:
```
この動画の背景を、都市の通りから宇宙ステーション内部に置き換えてください。
- 被写体（人物・衣服）は完全に保持
- 背景は金属とガラスの通路、青白いLED光、舷窓から見える地球
- 通路の床に微妙な無重力浮遊感（被写体の歩行は地上のまま）
- 環境光を青白に統一
- 被写体に冷色のリムライトを追加
- 元の音声に空調ノイズと低い機械音を重ねる
```

**コツ**: 「リムライト」指定で被写体がCG背景に馴染む。
**ネガティブ**: 被写体の歩行の不一致、現実的すぎる質感、温色光の残存

---

## 6. 背景のみ天候変更（晴れ → 雨）

**ユースケース**: 雰囲気の演出変更
**入力**: 晴天で撮影した屋外映像

**プロンプト**:
```
この動画の天候を、晴れから雨に変更してください。被写体（人物・衣服）は雨に濡れていない状態で保持してください（傘やひさしの下にいる想定）。
- 背景に降雨を追加（粒の大きさ中程度、密度中）
- 地面に水たまりと反射光
- 空を厚い灰色の雲に変更
- 全体のライティングを曇天の柔らかい拡散光に
- 環境音に雨音を控えめに追加
- 被写体は完全に保持（衣服は乾いたまま）
```

**コツ**: 「傘の下にいる想定」と書くと矛盾を解消できる。
**ネガティブ**: 被写体の衣服が濡れる、過剰な大雨、被写体への雨粒

---

## 7. 背景の人物群衆を削除（静寂化）

**ユースケース**: 喧騒のある場所を静かな空間に
**入力**: 観光地や駅で人混みがある映像

**プロンプト**:
```
この動画の背景にいる人物（メインの被写体以外）をすべて自然に削除してください。
- メインの被写体（中央の人物）のみ保持
- 削除した部分を背景の建物・道路・床で違和感なく埋める
- 削除部分の影や光の状態も補完
- カメラパンや被写体の動きには整合的に追従
- 元の被写体・カメラワーク・音声は保持
- 環境音から雑踏ノイズも軽減
```

**コツ**: ロケーションの「無人化」演出で映像の主役感が出る。
**ネガティブ**: ぼやけた人型のシルエット残存、背景の不自然な穴、繰り返しパターン

---

## 8. 背景のみ時間帯変更（昼 → 夜）

**ユースケース**: 撮り直し不要の昼夜変換
**入力**: 昼間に撮影した屋外映像

**プロンプト**:
```
この動画の時間帯を、昼から夜に変更してください。被写体は完全に保持してください。
- 空を満点の星空に変更
- 街灯や窓明かりを背景に追加（暖色の点光源）
- 被写体に冷色の月光リムライトを追加
- メインライトを月光相当の青白で再ライティング
- 影の方向を月の方向に再計算
- 全体の彩度を抑え、青みを足す
- 環境音から日中のノイズを除去し、虫の音や夜風を追加
```

**コツ**: 「リムライト」と「環境音」のセット指定で夜感が出る。
**ネガティブ**: 被写体への昼間光の残存、不自然なシルエット、過度な暗さ

---

## 9. 背景のみ季節変更（夏 → 冬）

**ユースケース**: 季節撮り直し不要の演出
**入力**: 夏に撮影した屋外映像

**プロンプト**:
```
この動画の季節を、夏から冬に変更してください。被写体（人物・衣服）は完全に保持してください（防寒着想定）。
- 背景の緑の木々を雪で覆われた裸木に変更
- 地面に積雪を追加（5cm程度）
- 空気中にちらつく雪片を控えめに追加
- 全体の色温度を冷色に
- 被写体の息を白く可視化
- 環境音から夏のセミ音を除去し、雪の上の靴音や冷たい風音を追加
```

**コツ**: 「息を白く」の演出が一気に冬感を強める。
**ネガティブ**: 夏の植物の残存、被写体の薄着、季節矛盾

---

## 10. 背景の建物を別の建築物に差し替え

**ユースケース**: ロケーション変更、版権回避
**入力**: 特定の建物前で撮影された映像

**プロンプト**:
```
この動画の背景にある建物を、別の建築様式の建物に差し替えてください。
- 被写体（人物）と前景は完全に保持
- 元の建物を、19世紀ヨーロッパ風の石造り建築に置換
- 建物の位置・距離・パースは元と一致
- カメラの動きに整合的に追従
- ライティングと影は元のシーンに整合
- 周辺の地面や前景物は変更しない
```

**コツ**: パース（遠近感）の一致が違和感解消の決定要因。
**ネガティブ**: 建物の浮き、パース矛盾、被写体への色被り
