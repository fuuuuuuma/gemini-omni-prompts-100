# 09. トランジション・継ぎ目編集 (Transition / Morph)

異なるクリップ間の切り替わりを編集する。シームレスな繋ぎや派手なエフェクト的なトランジションを生成。

---

## 1. シームレスモーフ（顔 → 別の顔）

**ユースケース**: ビフォーアフター、変身演出
**入力**: 2つの顔のクリップ（同じ画角・ライティング）

**プロンプト**:
```
この2つのクリップを、顔のシームレスモーフトランジションで繋いでください。
- クリップ1の顔から、クリップ2の顔へ滑らかに変形
- 変形時間は1秒
- 目・鼻・口の位置を補間しながら変化
- 髪の質感・服装も同様にモーフ
- 背景も同時にクロスフェード
- カット感のない自然な変形
- 元のライティングを維持
```

**コツ**: 同じ画角・ライティングのクリップを用意するとモーフが綺麗に決まる。
**ネガティブ**: パーツの歪み、急な変化、シルエットの矛盾

---

## 2. ホイップパン繋ぎ

**ユースケース**: ダイナミックなシーン転換
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、ホイップパン（高速横スワイプ）でトランジションしてください。
- クリップ1の末尾に高速な右方向パン（モーションブラー強）を生成
- パン中にクリップ2の冒頭にも同方向のパン
- パンの方向と速度を一致させる
- パン中はモーションブラーで内容が見えなくなる
- 全体のトランジションは0.3秒
- 効果音（風切り音）を控えめに追加可
```

**コツ**: 「同方向・同速度のパン」を両クリップに作るのがコツ。
**ネガティブ**: パン方向の不一致、ブラー不足、テンポの悪さ

---

## 3. オブジェクトワイプ（同色オブジェクトで切替）

**ユースケース**: 創造的なシーン切替
**入力**: 2つのクリップ + ワイプ用オブジェクト

**プロンプト**:
```
この2つのクリップを、画面を横切る赤い物体（ボール、車、布など）でワイプしながら切り替えてください。
- クリップ1の終わりに赤い物体が画面右端から左端へ通過
- 物体が画面を覆い隠した瞬間にクリップ2へ切替
- クリップ2でも物体が画面左から右へ抜けていく
- 物体のサイズと速度はトランジション全体で連続
- ライティングと色温度を整合
- カット感のないシームレスな切替
```

**コツ**: 「色」が同じ物体だと違和感なく繋がる。
**ネガティブ**: 物体の不連続、サイズの矛盾、切替タイミングのズレ

---

## 4. クロスディゾルブ（自然な被せ）

**ユースケース**: ドキュメンタリー、回想シーン
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、滑らかなクロスディゾルブで繋いでください。
- ディゾルブ時間は1秒
- クリップ1のフェードアウトとクリップ2のフェードインを重ねる
- 中間点で両方が50%ずつ重なる
- 色温度と明るさのギャップを補正して自然に
- 音声もクロスフェード
- ディゾルブ中の被写体の動きは両方とも保持
```

**コツ**: 古典的だが安定。「明るさギャップ補正」が品質の鍵。
**ネガティブ**: ディゾルブの不自然な濁り、明るさの段差

---

## 5. ズームトランジション

**ユースケース**: テンポの良いVlog
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、ズームイントランジションで繋いでください。
- クリップ1の末尾を急速にズームイン（被写体を中心に5倍まで）
- ズーム最大時に画面が白飛びまたはぼやけ
- そこからクリップ2の冒頭にズームアウト
- 全体のトランジションは0.4秒
- モーションブラーを強めに付与
- 音声は短いスウィッシュ音を追加可
```

**コツ**: VlogやTikTok編集の定番。スピード感が出る。
**ネガティブ**: ズームの遅さ、ブラー不足、テンポ感の欠如

---

## 6. マスクトランジション（形状抜き）

**ユースケース**: 創造的なシーン切替
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、円形マスクトランジションで繋いでください。
- 画面中央に小さな円が出現
- 円の内部にはクリップ2が表示
- 円が画面全体を覆うまで拡大
- 拡大は0.5秒、イーズアウト
- 円のエッジは柔らかいフェザー処理
- 円の中心は被写体の顔か画面中央
- 拡大中、円の外側はクリップ1のまま
```

**コツ**: 円以外にも「五角形」「アイコン形」など指定可。
**ネガティブ**: 円のエッジのジャギー、急な拡大、中心ズレ

---

## 7. グリッチトランジション

**ユースケース**: テック系、サイバーパンク
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、デジタルグリッチトランジションで繋いでください。
- クリップ1の末尾に色収差・ピクセル乱れ・走査線ノイズを徐々に追加
- グリッチがピークに達した瞬間にクリップ2へ切替
- クリップ2の冒頭でグリッチが徐々に収まる
- 全体のトランジションは0.6秒
- 効果音（デジタルノイズ）を追加
- 色チャンネルのRGB分離も加える
```

**コツ**: 「RGB分離」が一気にグリッチ感を演出。
**ネガティブ**: グリッチの過剰、視認性の悪化、ノイズ感不足

---

## 8. リキッドモーフ（液体変形）

**ユースケース**: 創造的、アート系
**入力**: 2つのクリップ

**プロンプト**:
```
この2つのクリップを、液体のように変形するリキッドモーフトランジションで繋いでください。
- クリップ1が液状化して波打ちながら溶ける
- 液体が一度集まり、新しい形に再構成されてクリップ2に
- 全体のトランジションは1秒
- 液体の表面張力、反射、流動性を表現
- 色は両クリップの中間色で滑らかに変化
- カット感のない有機的な変形
```

**コツ**: 「表面張力・反射」など物性指定で写実度が上がる。
**ネガティブ**: 単純なディゾルブ感、液体感不足、不自然な変形

---

## 9. ロケーション移動風トランジション

**ユースケース**: 物語的な「場面移動」演出
**入力**: 2つの異なるロケーションのクリップ

**プロンプト**:
```
この2つのクリップを、被写体がロケーションを瞬間移動するトランジションで繋いでください。
- 被写体は両クリップで同じ人物・同じ服装
- クリップ1で被写体が一歩踏み出す瞬間に画面全体が光る
- 光がピークに達した瞬間にクリップ2へ
- クリップ2では被写体が新ロケーションで一歩踏み出している
- 被写体の動きの連続性を維持
- 全体のトランジションは0.5秒
```

**コツ**: 「一歩踏み出す」など動きを揃えるとマジック感が出る。
**ネガティブ**: 被写体の動きの不一致、光の不自然な演出

---

## 10. ライティング切替トランジション

**ユースケース**: 時間帯変化、ムード切替
**入力**: 同じシーンの2つのクリップ（異なるライティング）

**プロンプト**:
```
この2つのクリップを、ライティングが滑らかに変化するトランジションで繋いでください。
- クリップ1の照明状態から、クリップ2の照明状態へ徐々に変化
- 光源の方向・色温度・強度が滑らかに補間
- 被写体と背景の位置・構図は維持
- 全体のトランジションは1秒
- 影の方向と長さも整合的に変化
- カット感のないシームレスな照明変化
```

**コツ**: 同じ構図で照明だけ変えると「時間の経過」演出になる。
**ネガティブ**: 急な光の変化、被写体の動き、影の不整合
