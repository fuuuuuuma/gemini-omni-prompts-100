# 07. カラーグレーディング (Color Grading / LUT)

映像全体のカラートーンを調整して、映画的・特定ジャンル的なムードを作る。LUT適用相当。

---

## 1. シネマティック・オレンジ&ティール

**ユースケース**: ハリウッド映画調の質感
**入力**: 平坦な色調の映像

**プロンプト**:
```
この動画にハリウッド映画調のオレンジ&ティール（青緑）グレーディングを適用してください。
- シャドウとミッドトーンの一部にティール（青緑）を載せる
- ハイライトと肌のトーンにオレンジを載せる
- 全体のコントラストを上げる
- 黒は完全な黒に近く沈める
- 白は飛ばさず、わずかに彩度を残す
- フィルム的なソフトなロールオフ
- 元の構図・動き・音声を保持
```

**コツ**: 「肌をオレンジ・影をティール」が王道の対比。
**ネガティブ**: 過剰な青味、肌の不自然な発色、色の濁り

---

## 2. ブレードランナー風ネオン

**ユースケース**: SF・サイバーパンク演出
**入力**: 夜の都市映像

**プロンプト**:
```
この動画をブレードランナー2049風のネオンカラーグレーディングに変換してください。
- マゼンタとシアンの2色を強調
- 全体は深い黒寄りの暗部に
- ハイライトに鮮やかなマゼンタかシアンのグロー
- 中間トーンは沈めて重厚感
- 大気感のあるソフトな霧効果
- 微細な色収差をハイライトに付与
- 元の構図と動きを保持
```

**コツ**: 「2色のみ強調」と限定すると統一感が出る。
**ネガティブ**: 自然光の残存、過剰な色数、白浮き

---

## 3. 北欧映画調（冷色寄り）

**ユースケース**: 静謐な物語、ミステリー
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画を北欧映画調のクールなグレーディングに変換してください。
- 全体の色温度を下げ、青〜緑寄りに
- 彩度を全体的に抑える（実写比60%）
- ハイライトは白に近い淡いブルー
- シャドウは沈むがニュートラルブラックではなく濃紺寄り
- フィルム的なソフトな質感
- 肌のトーンも温かみを抑える
- 元の構図と動きを保持
```

**コツ**: 彩度を「60%」で指定するとコントロール可能。
**ネガティブ**: 過剰な彩度、温色の残存、コントラストの過剰

---

## 4. ヴィンテージフィルム（コダクローム風）

**ユースケース**: レトロ感のあるドキュメンタリー、回想
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画を1970年代のコダクローム風カラーグレーディングに変換してください。
- 鮮やかな赤と黄、深い緑を強調
- 青はやや沈める
- フィルム特有の柔らかいハイライトロールオフ
- 軽いグレイン（粒子）を全体に
- 黒は深く沈むが、ニュートラルではなく茶色寄り
- 全体に温かみのあるトーン
- 元の構図・動き・音声を保持
```

**コツ**: 「赤・黄・緑を強調、青を抑える」がコダクロームの特徴。
**ネガティブ**: デジタル感、青の強い色、シャープすぎる解像感

---

## 5. ハイコントラスト・モノクロ

**ユースケース**: アート系PV、ファッション
**入力**: カラー映像

**プロンプト**:
```
この動画をハイコントラストのモノクロ映像に変換してください。
- 完全なモノクロ（着色なし）
- 黒は完全な黒、白は飛ばし気味
- 中間グレーは少なく、白と黒のメリハリ
- 肌のトーンは赤チャンネル強調で滑らかに
- 軽い35mmフィルムグレイン
- 元の構図・動き・音声を保持
- アスペクト比はそのまま
```

**コツ**: 「赤チャンネル強調」で肌の質感が美しく出る。
**ネガティブ**: セピア、グレー過多、コントラスト不足

---

## 6. パステルトーン（韓国ドラマ風）

**ユースケース**: 恋愛系・青春系コンテンツ
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画を韓国ドラマ風のパステルトーン・カラーグレーディングに変換してください。
- 全体の彩度を抑え、淡いパステル調に
- ハイライトに淡いピンクと薄い水色
- 肌のトーンは滑らかでマットな質感
- 黒は完全な黒ではなく、淡いネイビー
- 全体にソフトな白い霞（フォグ）を薄く
- 元の構図と動きを保持
```

**コツ**: 「淡い・滑らか・マット」のキーワードで再現性が上がる。
**ネガティブ**: 高彩度、ハイコントラスト、シャープな質感

---

## 7. ティーン映画調（鮮やかな彩度）

**ユースケース**: 青春コメディ、ファッションコンテンツ
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画をアメリカン・ティーン映画調の鮮やかなカラーグレーディングに変換してください。
- 全体の彩度を150%に強調
- 赤・青・黄など原色系を特に強める
- コントラストは中程度
- 肌のトーンは健康的なオレンジ寄り
- ハイライトに微妙なグロー
- 全体に明るくポジティブな雰囲気
- 元の構図と動きを保持
```

**コツ**: 「健康的なオレンジ寄り」の肌が爽やか感の核。
**ネガティブ**: 暗い影、青白い肌、低彩度

---

## 8. ホラー映画調（緑がかった暗色）

**ユースケース**: ホラー・サスペンス
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画をモダンホラー映画調のグレーディングに変換してください。
- 全体の色を緑がかった暗色に
- シャドウを深く沈める（黒は青緑寄り）
- ハイライトは抑え目で、わずかに青み
- 肌のトーンを血色のない蒼白に
- 中間トーンを濁らせる
- 微細なフィルムグレイン
- 不気味な雰囲気
- 元の構図と動きを保持
```

**コツ**: 「血色のない蒼白」の肌で一気にホラー感が出る。
**ネガティブ**: 温色、明るい肌色、彩度の高さ

---

## 9. ドキュメンタリー調（フラット・自然）

**ユースケース**: 報道・教育コンテンツ
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画をドキュメンタリー調のフラットで自然なカラーグレーディングに変換してください。
- 過度な色付けをせず、自然な色温度に補正
- コントラストはやや控えめ
- 肌のトーンを健康的かつ自然に
- 黒は完全に沈めず、暗部のディテールを残す
- ハイライトも飛ばさず情報を残す
- フィルム的な特殊効果なし
- 元の構図と動きを保持
```

**コツ**: 「情報を残す」（HDR的な思想）でドキュメンタリーの品位が出る。
**ネガティブ**: 過剰なルック、シネマティックなコントラスト、特殊エフェクト

---

## 10. ラグジュアリーCM調（ゴールド寄り）

**ユースケース**: 高級ブランド、化粧品、宝石
**入力**: 通常撮影の映像

**プロンプト**:
```
この動画を高級ブランドCM調のゴールド寄りカラーグレーディングに変換してください。
- ハイライトに金色のグロー
- 中間トーンを暖色寄りに（オレンジ・ゴールド）
- シャドウは深い茶色（黒よりも暖かい）
- 全体に上品なソフトコントラスト
- 肌のトーンは滑らかでツヤ感
- 微妙なゴールドの光漏れ（レンズフレア）
- 全体的に重厚で華やか
- 元の構図と動きを保持
```

**コツ**: 「ゴールドの光漏れ」が高級感の決め手。
**ネガティブ**: 冷色、コントラストの粗さ、安っぽい鮮やかさ
