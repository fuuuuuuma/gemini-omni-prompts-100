# 01. スタイル変換 (Style Transfer)

実写映像のスタイルを別の絵柄・質感に変換する。元の動き・構図・タイミングは保持しつつ、テクスチャ・カラー・ストロークを別のスタイルに置き換える。

---

## 1. 実写Vlog → Studio Ghibli風アニメ

**ユースケース**: 旅行Vlogの導入をアニメ調OPとして使う
**入力**: 屋外を歩く実写映像（5〜10秒推奨）

**プロンプト**:
```
この動画をスタジオジブリ風の手描きアニメーションに変換してください。元の動き、人物のポーズ、カメラアングルは完全に維持してください。
- 背景は柔らかい水彩テクスチャを適用
- 肌のトーンはフラットなセル画風カラーリング
- 表情と口の動きはそのまま保持
- フレームレートと尺は元のまま
- 髪と衣服にわずかな風の効果を追加
- 線画は手描き感のある太さの揺らぎを持たせる
```

**コツ**: 「元の動きを完全に維持」を明示するとアニメ化時の動き破綻を防げる。
**ネガティブ**: フォトリアルな質感、3DCGレンダリング、プラスチック感のある肌、CGアニメ調

---

## 2. 実写 → Pixar風3D CGアニメーション

**ユースケース**: ファミリー向けPVや絵本風映像
**入力**: 人物のクローズアップ実写（3〜8秒）

**プロンプト**:
```
この動画をPixar風の3D CGアニメーションに変換してください。元の表情、視線の動き、口元のリップシンクは正確に保持してください。
- 大きな目と柔らかい輪郭の頭身デフォルメ（実写比 1.2倍頭身）
- サブサーフェススキャタリングを効かせた肌の質感
- 衣服はマットな布シミュレーション
- 背景はソフトフォーカスでバブルライト
- 全体のライティングは温かいキーライト + クールフィル
```

**コツ**: 頭身デフォルメ比を数値で指定すると過剰変形を防げる。
**ネガティブ**: ホラー調の不気味な顔、過度なデフォルメ、平面的なセル画

## 3. 実写 → 水彩画風アニメーション

**ユースケース**: ドキュメンタリーや回想シーンの挿入カット
**入力**: 静かなシーンの実写（風景や人物）

**プロンプト**:
```
この動画を透明感のある水彩画スタイルのアニメーションに変換してください。元の構図とカメラワークは保持してください。
- 色は淡い顔料が紙にじむような滲み表現
- 輪郭線はインクのにじみで描かれる
- 動きはフレームごとに微妙な筆致の揺らぎ（30%）
- 背景は粗い水彩紙テクスチャ
- 彩度は実写比60%まで下げる
```

**コツ**: 筆致の揺らぎは20〜40%の範囲が自然。100%にすると視聴困難。
**ネガティブ**: デジタル感のあるベタ塗り、シャープな線、写真的なディテール

## 4. 実写 → フィルムノワール（白黒映画）調

**ユースケース**: ミステリー作品や芸術系作品の導入
**入力**: 室内で人物が動く実写（5〜15秒）

**プロンプト**:
```
この動画を1940年代フィルムノワール調のモノクロ映像に変換してください。元の動きと演技は保持してください。
- 完全モノクロ（白黒2階調ではなく豊かなグレースケール）
- ハイコントラスト（影は深く沈み、ハイライトは飛び気味）
- ベネチアンブラインドの影模様を被写体に投影
- 16mmフィルムグレインを強めに付与
- アスペクト比は1.85:1にレターボックス化
```

**コツ**: ブラインド影の追加で一気にノワール感が出る。
**ネガティブ**: セピアトーン、カラー残し演出、現代的なクリーンさ

## 5. アニメ → 実写化（フォトリアル変換）

**ユースケース**: アニメ作品の実写プロモーション映像
**入力**: アニメーション映像クリップ（5〜10秒）

**プロンプト**:
```
このアニメーション映像をフォトリアルな実写映像に変換してください。元の動き、ポーズ、表情、カメラワークは完全に保持してください。
- キャラクターを実在しそうな人物として再構成
- 肌・髪・衣服はすべて実在の物理素材として描写
- 背景は実際のロケーションとして再構築
- ライティングは自然光ベースで一貫
- フィルムカメラ（ARRI Alexa風）で撮影されたような質感
```

**コツ**: 「実在しそうな人物」と書くことで版権キャラの直接コピーを避ける。
**ネガティブ**: アニメ的な大きな目、極端な髪色、平面的なセル画感

## 6. カラー映像 → ヴィンテージ8mmフィルム

**ユースケース**: レトロ感のあるホームムービー演出
**入力**: スマホ撮影のホームビデオ

**プロンプト**:
```
この動画を1970年代の8mmフィルムで撮影されたようなヴィンテージ映像に変換してください。元の動きと内容はすべて保持してください。
- 強いフィルムグレイン（粒子感）
- 退色した色（赤と黄が強く、青が弱い）
- フレーム端のビネット（暗角）
- 微細な傷とホコリのちらつき
- フレームレート 18fps相当のかすかなカクつき
- フレームジッターを5%混入
```

**コツ**: 18fps相当 + ジッター5%で一気にホームムービー感が出る。
**ネガティブ**: 4K感のあるシャープさ、現代的な彩度、デジタルノイズ

## 7. 実写 → サイバーパンクネオン調

**ユースケース**: テックチャンネルのOP、ガジェット紹介
**入力**: 夜の都市や室内の実写（5〜10秒）

**プロンプト**:
```
この動画をサイバーパンク2077風のネオン都市スタイルに変換してください。元の動きとカメラワークは保持してください。
- 主要光源をマゼンタとシアンの2色ネオンに置換
- 雨や霧を背景に追加（地面に反射）
- 看板やディスプレイをグリッチした漢字・ハングル表示に
- 黒の階調を深く沈ませ、ハイライトをグロー
- 大気感のあるボリュメトリックライトを追加
```

**コツ**: 「マゼンタとシアン」と色を限定するとブレない。
**ネガティブ**: 自然光、パステルトーン、明るい昼間の雰囲気

## 8. 実写 → 油絵風アニメーション

**ユースケース**: 美術系チャンネル、芸術PV
**入力**: 静物・風景・人物の実写

**プロンプト**:
```
この動画をゴッホ風の油絵タッチのアニメーションに変換してください。元のフレーミングと動きは保持してください。
- 厚塗り（インパスト）の絵の具質感
- うねるような渦巻き状の筆致
- 彩度は高く、相補色のコントラストを強調
- フレームごとに筆致が微妙に変化する手描きアニメ感
- 背景の空には大きなブラシストロークの渦巻き
```

**コツ**: 画家名（ゴッホ／モネ／セザンヌ）を指定すると方向性が明確。
**ネガティブ**: 写実的な描写、デジタル的な平滑さ、薄塗りの水彩感

## 10. 実写 → Unreal Engineゲーム風レンダリング

**ユースケース**: ゲーム実況のOP、テックデモ
**入力**: 人物や乗り物の実写

**プロンプト**:
```
この動画をUnreal Engine 5でリアルタイムレンダリングしたゲーム映像のように変換してください。元の動きとカメラワークは保持してください。
- Lumenグローバルイルミネーション風の柔らかい間接光
- Nanite的な超高密度ジオメトリ感
- 被写体に微妙なリムライト
- 背景はフォトグラメトリで取り込んだような細密度
- レイトレース反射が水面・金属に乗る
- 画面端にわずかな色収差
```

**コツ**: 「Lumen」「Nanite」など実技術名を入れると質感の方向性が明確。
