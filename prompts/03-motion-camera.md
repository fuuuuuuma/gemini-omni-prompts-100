# 03. モーション・カメラワーク制御 (Motion / Camera Control)

静止ショットに動きをつけたり、既存のカメラワークを別のワークに置き換える。後付けでシネマティックな表現を加える。

---

## 1. 静止ショット → ドリーイン（前進寄せ）

**ユースケース**: 三脚固定の地味な引き画に動きを後付け
**入力**: 動きの少ない固定ショット（3〜8秒）

**プロンプト**:
```
この動画にゆっくりとシネマティックなドリーイン（前進カメラ移動）を適用してください。
- 開始時は元のフレーミング
- 終了時は被写体の顔を中心に35%寄った構図
- 背景レイヤーに微妙なパララックス（視差）を追加
- 元の音声と被写体の動きはすべて保持
- スムーズなイーズイン・イーズアウト、ジッターやブレなし
- 元のアスペクト比を維持
```

**コツ**: 「35%寄った」のように数値で始点/終点を固定。
**ネガティブ**: 手ブレ、急なズーム、被写体へのモーションブラー

---

## 2. 静止ショット → ドリーアウト（後退引き）

**ユースケース**: クライマックスから引いて余韻を演出
**入力**: クローズアップの静止ショット

**プロンプト**:
```
この動画にゆっくりとしたドリーアウト（後退カメラ移動）を適用してください。
- 開始時は元のフレーミング
- 終了時は元の構図から40%引いた広い画
- 引いた画で周辺のロケーション情報が見えるようにアウトペイント
- 背景のパララックスで奥行きを強調
- 元の音声と被写体の動きを保持
- イーズアウトで余韻を持たせる
```

**コツ**: ドリーアウトは「全体像が見える」演出に有効。
**ネガティブ**: 急な引き、ジャンプカット、被写体のブレ

---

## 3. 横パン（左→右）を追加

**ユースケース**: 風景や群衆の全体像を見せる
**入力**: 広めの静止ショット

**プロンプト**:
```
この動画にゆっくりとした横パン（カメラを左から右へ水平回転）を適用してください。
- 全体の尺で約30度のパン角度
- 等速ではなくイーズイン・イーズアウト
- パン中も被写体や前景の動きは保持
- パンに伴う見切れ部分はアウトペイントで補完
- 元の音声を維持
```

**コツ**: パン角度を「30度」で指定すると過剰な振りを防げる。
**ネガティブ**: 急な振り、手ブレ、被写体の歪み

---

## 4. 縦ティルト（下→上）を追加

**ユースケース**: 高層建築や全身ショットの紹介
**入力**: 静止した広角ショット

**プロンプト**:
```
この動画にゆっくりとした縦ティルト（カメラを下から上へ垂直回転）を適用してください。
- 開始時は被写体の足元・基部
- 終了時は被写体の頭上・上空
- 全体の尺で約45度のティルト角度
- イーズイン・イーズアウト
- 見切れる上下のフレームはアウトペイントで補完
- 元の音声を維持
```

**コツ**: 建築物や人物の「全身紹介」演出に有効。
**ネガティブ**: 急な振り、被写体のパース歪み

---

## 5. オービット（被写体周回）カメラを追加

**ユースケース**: 商品紹介、被写体を立体的に見せる
**入力**: 中央に被写体がある静止ショット

**プロンプト**:
```
この動画に被写体を中心としたオービット（周回）カメラワークを適用してください。
- 被写体を中心軸に、カメラを右回りに90度周回
- 周回中も被写体は常に画面中央に保持
- 周回に伴う背景の見え方の変化は整合的にアウトペイント
- 周回速度はイーズイン・イーズアウトで滑らか
- 被写体への距離・画角は一定
- 元の音声と被写体の動きを保持
```

**コツ**: 商品紹介や人物プロフィール映像で有効。
**ネガティブ**: 周回中の被写体ブレ、画角変動、不自然な背景

---

## 6. クレーンショット（上昇カメラ）を追加

**ユースケース**: シーン全体の俯瞰、エンディング演出
**入力**: アイレベルの静止ショット

**プロンプト**:
```
この動画にクレーンショット（カメラの垂直上昇）を適用してください。
- 開始時はアイレベル（人物の目線高さ）
- 終了時は高度約5m、被写体を見下ろす角度
- 上昇に応じてアングルが徐々に俯瞰へ変化
- 周辺ロケーションが見えるよう上空をアウトペイント
- イーズアウトで余韻を持たせる
- 元の音声を維持
```

**コツ**: エンディングの「離れていく」演出で使うと効果的。
**ネガティブ**: 急な上昇、被写体の歪み、画角変動

---

## 7. 手ブレ除去（スタビライズ）

**ユースケース**: スマホ撮影の手ブレ映像を三脚撮影風に
**入力**: 手持ち撮影の手ブレ映像

**プロンプト**:
```
この手ブレのある動画から手ブレを完全に除去し、三脚で撮影したような静止ショットに変換してください。
- 全フレームを共通基準に揃えてスタビライズ
- 残存する微細な揺れも除去
- フレーム端の見切れる部分はアウトペイントで補完
- 元のカメラの大局的な動き（パンやティルト）は保持
- 被写体の動きや音声は変えない
```

**コツ**: 「大局的な動きは保持」を入れないと意図したカメラワークまで消える。
**ネガティブ**: 過剰なクロップ、被写体のブラー、不自然な歪み

---

## 8. 手ブレ感を追加（VLOG風）

**ユースケース**: 三脚映像をVLOG風の生っぽさに変換
**入力**: 三脚で撮影した安定した映像

**プロンプト**:
```
この三脚撮影の動画に、自然な手持ち撮影の手ブレ感を追加してください。
- 微細なランダムな手ブレ（振幅 ±3px）
- 歩行リズムに合わせた上下の揺れ（毎秒 約1回）
- 呼吸的な微振動
- 大局的なカメラの動きは元のまま維持
- 被写体の動きや音声は変えない
- VLOG動画の生っぽさを演出
```

**コツ**: 振幅を「±3px」で指定すると過剰な揺れを防げる。
**ネガティブ**: 酔いそうな大揺れ、リズムのない不規則ブレ

---

## 9. スローズーム（5秒で1.2倍）

**ユースケース**: ドキュメンタリーの語り場面
**入力**: 静止インタビューショット

**プロンプト**:
```
この動画に、ほとんど気づかれない速度のスローズームインを適用してください。
- 5秒で1.2倍まで等速に近いゆるやかなズーム
- 被写体の顔は常に画面中央に保持
- 動きは「ほぼ静止」と感じられるほど微細
- 元の音声と被写体の動きを完全に保持
- ジッターやブレなし
```

**コツ**: ドキュメンタリーの感情高まり演出で定番。
**ネガティブ**: 気づくほどの速いズーム、被写体のフレームアウト

---

## 10. 被写体追従カメラ（トラッキングショット）

**ユースケース**: 走る人物や動く乗り物の追従
**入力**: 動く被写体を引いて撮った映像

**プロンプト**:
```
この動画のカメラを、移動する被写体に追従するトラッキングショットに変換してください。
- 被写体を常に画面中央〜やや前方に保持
- 被写体の速度に合わせてカメラも横移動
- 背景は被写体の進行方向と逆向きに流れる
- 適度なモーションブラーを背景にのみ付与
- 被写体はシャープに保つ
- 元の音声を維持
```

**コツ**: 「被写体はシャープ・背景のみブラー」が追従感の鍵。
**ネガティブ**: 被写体のブラー、背景の静止、画角の急変
