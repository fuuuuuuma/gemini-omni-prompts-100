# 08. テンポ・速度操作 (Speed / Time Remap)

動画の再生速度を変更したり、一部のみ可変速にしたりする。時間軸を編集する。

---

## 1. 全編スローモーション（0.5倍速）

**ユースケース**: アクションシーンの強調、エモーショナル演出
**入力**: 通常速度の動画

**プロンプト**:
```
この動画を全編0.5倍速のスローモーションに変換してください。
- フレーム補間で滑らかに2倍の尺に
- 中間フレームをAI補間生成（モーションブラーは抑制）
- 音声はピッチを保持したまま2倍に引き伸ばすか、無音に
- 被写体の動きの自然な連続性を維持
- 背景の動きも整合的に減速
- ループのないシームレスな減速
```

**コツ**: 「ピッチ保持」を指定しないと声が不気味に低くなる。
**ネガティブ**: 静止フレームの繰り返し、ジッター、モーションブラーの過剰

---

## 2. 全編ハイパーラプス（4倍速）

**ユースケース**: 風景・タイムラプス的演出
**入力**: 通常速度の動画

**プロンプト**:
```
この動画を全編4倍速のハイパーラプスに変換してください。
- 全体の尺を1/4に圧縮
- フレーム間引きではなく、AI圧縮で滑らかさを維持
- 適度なモーションブラーを動きの大きい部分に付与
- 被写体の連続性を維持
- 音声は無音化、もしくはハイパーラプス用BGMに置換可
- カメラの揺れがあれば軽くスタビライズ
```

**コツ**: 「モーションブラーで滑らかさ」が高速感の鍵。
**ネガティブ**: カクついた早送り、被写体の消失、急な視点変化

---

## 3. 部分スロモ（クライマックスのみ）

**ユースケース**: 決定的瞬間の強調
**入力**: 短いアクション映像

**プロンプト**:
```
この動画の中央部（最も動きの大きい瞬間）の1秒間のみ、0.3倍速の部分スローモーションに変換してください。
- 前後は通常速度
- スローモーション部分への突入・脱出は滑らかなランプ
- スローモーション中のフレームはAI補間
- 元の音声は通常速度部分でのみ再生、スロー部分は無音または効果音
- 被写体の動きの連続性を維持
- カット感のない滑らかな速度変化
```

**コツ**: 「突入・脱出のランプ」が職人的な編集感を出す。
**ネガティブ**: 急なカット、速度の不連続、被写体のブレ

---

## 4. ランプスロー（速 → 遅へ滑らかに）

**ユースケース**: エンディングの余韻演出
**入力**: 通常速度の動画

**プロンプト**:
```
この動画に、冒頭から末尾にかけて徐々に遅くなるランプスローを適用してください。
- 冒頭: 1.0倍速（通常）
- 末尾: 0.3倍速
- 全体の速度カーブはイーズアウト
- AI補間で滑らかなフレーム生成
- 音声もピッチ保持で同様にランプ
- 被写体の動きの連続性を維持
- 余韻のある終わり方
```

**コツ**: エンディングの「時間が止まる」演出に最適。
**ネガティブ**: 急な速度変化、カクつき、音のピッチ崩れ

---

## 5. リバース再生

**ユースケース**: 巻き戻し演出、コミカル表現
**入力**: 通常速度の動画

**プロンプト**:
```
この動画を完全に逆再生に変換してください。
- 全フレームを末尾→冒頭の順で再生
- 被写体の動きが逆向きに見える
- フレーム間の動きの連続性を維持
- 音声は無音化、もしくは逆再生BGMに置換可
- 軽い「巻き戻し感」のあるエフェクト（オプション）
- 元のカメラワークもそのまま逆再生
```

**コツ**: シンプルな指示でも効果絶大。
**ネガティブ**: 音声の逆再生による違和感（無音推奨）

---

## 6. フリーズフレーム（特定瞬間を静止画化）

**ユースケース**: 決定的瞬間の強調、テキスト挿入
**入力**: 動きのある動画

**プロンプト**:
```
この動画の中央フレーム（最も視覚的にインパクトのある瞬間）で2秒間フリーズフレームを挿入してください。
- フリーズ前: 通常再生
- 中央: 静止画として2秒間保持
- フリーズ後: 通常再生に復帰
- フリーズ突入時に軽いズームインを付与
- フリーズ中、被写体の周りに微妙なグロー
- 音声はフリーズ中のみ低音BGMまたは無音
- フリーズ脱出は滑らかに
```

**コツ**: 「軽いズームイン」「グロー」で物語的に強調できる。
**ネガティブ**: 急なフリーズ突入、画面のカクつき

---

## 7. インターバル撮影風（タイムラプス変換）

**ユースケース**: 風景・建設工程の時短演出
**入力**: 長時間の動画

**プロンプト**:
```
この動画をタイムラプス（インターバル撮影）風に変換してください。
- 1秒あたり1〜2フレームに間引き、4〜8倍速圧縮
- カメラは三脚固定相当のスタビライズ
- 被写体（雲・人物・光）の動きが早回しに
- 適切なモーションブラーを動きに付与
- 音声は無音化、または環境音BGMに置換
- 風景や建築物の経時変化が強調される
```

**コツ**: タイムラプス映像感を出すには「スタビライズ + 大きなモーションブラー」が鍵。
**ネガティブ**: 手ブレ、被写体のジッター、不自然なカット感

---

## 8. ビートシンク（音楽に合わせた可変速）

**ユースケース**: MV・SNS向けリズム編集
**入力**: 動画 + 音楽トラック

**プロンプト**:
```
この動画を、添付した音楽のビートに同期した可変速編集に変換してください。
- ビートの強拍に合わせて映像のリズミカルな速度変化
- 強拍直前にスローダウン、強拍で通常速度に戻る
- 速度変化のピーク差は0.5倍〜1.5倍
- フレーム間の連続性を維持
- 音楽は変更しない（速度キープ）
- 全体の尺は音楽尺に合わせる
- カット感のない滑らかな可変速
```

**コツ**: ビート前のスローダウンが「タメ」の演出に。
**ネガティブ**: ビートとのズレ、急なカット、過剰な速度変化

---

## 9. ブーメラン（正再生 → 逆再生）

**ユースケース**: Instagram Reels、楽しいループ
**入力**: 短いクリップ（1〜3秒）

**プロンプト**:
```
この動画をブーメラン形式に変換してください。
- 元の動画を1回正再生
- そのまま逆再生でつなぐ
- 全体の尺は元の2倍
- 正→逆の継ぎ目をシームレスに
- 音声は無音化推奨（ブーメラン音楽に差し替え可）
- ループしてもまた繋がる
- 被写体の動きの大筋を保持
```

**コツ**: ループ動画として無限再生できる作り。
**ネガティブ**: 継ぎ目の不自然さ、音声の逆再生違和感

---

## 10. スピードランプ（特定アクションを強調）

**ユースケース**: スポーツ・アクションのハイライト
**入力**: アクション映像

**プロンプト**:
```
この動画にスピードランプ編集を適用してください。
- アプローチ部分: 1.5倍速（高速）
- アクション瞬間（中央0.5秒）: 0.25倍速（超スロー）
- 余韻部分: 1.0倍速（通常）
- 速度変化はすべて滑らかなランプ
- AI補間でフレーム生成
- 音声はピッチ保持で同期
- 動きの連続性を維持
- スポーツ的なダイナミックさ
```

**コツ**: 「速→超スロー→通常」の3段階がアクション映像の王道。
**ネガティブ**: 急な速度カット、ジッター、音声のピッチ崩れ
