# 06. 人物編集 (Character Edit)

被写体の人物属性（服装・髪型・表情・体型・年齢など）を編集する。

---

## 1. 服装変更（カジュアル → ビジネススーツ）

**ユースケース**: 同じ素材でフォーマル/カジュアル両用展開
**入力**: カジュアル服で話している人物映像

**プロンプト**:
```
この動画の被写体が着ている服を、Tシャツから紺色のビジネススーツ（ネクタイ付き）に変更してください。
- 顔・髪型・体型・ポーズは完全に保持
- スーツのフィット感は自然（体型に沿う）
- 動作（腕の動き・呼吸）に応じて布の動きも整合
- 元のライティングに整合した質感
- 影は体に応じて自然に投影
- 背景は変更しない
```

**コツ**: 「フィット感は自然」を入れると不自然なダボつき・ピチピチを防げる。
**ネガティブ**: 服の浮き、サイズ不一致、不自然な皺

---

## 2. 髪型・髪色変更

**ユースケース**: ヘアサロンのシミュレーション、キャラ変
**入力**: 人物のクローズアップ映像

**プロンプト**:
```
この動画の被写体の髪型を、ショートヘアからウェーブのかかったロングヘア（栗色）に変更してください。
- 顔・表情・体・服装は完全に保持
- 髪のボリュームと長さは肩より下まで
- 動作に応じて髪が自然に揺れる
- ライティングを髪の質感に再計算（ハイライトとシャドウ）
- 影は背景や肩に自然に投影
- フレーム間で安定
```

**コツ**: 髪の動きの追従が違和感解消の鍵。
**ネガティブ**: 髪の浮き、不自然な静止、頭部のサイズ変化

---

## 3. 表情変更（無表情 → 笑顔）

**ユースケース**: NG表情の差し替え
**入力**: 無表情で話している人物映像

**プロンプト**:
```
この動画の被写体の表情を、無表情から自然な笑顔に変更してください。
- 口角は軽く上がり、目尻も連動して微妙に細める
- 口の動き（リップシンク）は元の発話と一致
- 顔の輪郭・髪型・服装は完全に保持
- フレーム間で表情が滑らかに連続
- 元のライティングを保持
- 過剰な笑顔ではなく、ナチュラルな微笑
```

**コツ**: 「ナチュラルな微笑」と指定すると気持ち悪い笑顔を防げる。
**ネガティブ**: 不自然な歯の露出、目の死、表情のちらつき

---

## 4. 体型調整（自然な範囲で）

**ユースケース**: ファッション動画の体型補正
**入力**: 全身が映る人物映像

**プロンプト**:
```
この動画の被写体の体型を、自然な範囲で5%スリム化してください。
- 顔・髪型・服装の色とデザインは完全に保持
- ウエストと脚のラインを微妙に細くする
- 服のフィット感は体型に応じて自然に再調整
- 動作と歩行のリズムは保持
- 不自然な細さや骨格の歪みを避ける
- フレーム間で安定
```

**コツ**: 「5%」など小さい数値で控えめに。過剰な改変は不自然。
**ネガティブ**: 骨格の歪み、衣服のフィット崩れ、不自然な細さ

---

## 5. 顔の入れ替え（参考画像から）

**ユースケース**: スタンドイン俳優の顔置換（本人同意済み素材のみ）
**入力**: 動画 + 入れ替え対象の顔参考画像

**プロンプト**:
```
この動画の被写体の顔を、添付した参考画像の人物の顔に置き換えてください（本人同意済み素材を前提とします）。
- 表情・口の動き・視線方向は元の動画を完全に保持
- 髪型・服装・体型は元の動画を保持
- 顔の輪郭・肌の質感・髪の生え際を自然に統合
- ライティングを元のシーンに完全整合
- フレーム間で顔の安定性を確保（揺れ・歪みなし）
- 表情の微細な変化（瞬き・口角）も整合的に再生成
```

**コツ**: 「同意済み素材」を必ず明記。倫理的配慮の意思表示。
**ネガティブ**: 顔の浮き、輪郭のハロー、表情のちらつき

---

## 6. 年齢変更（若返り・老化）

**ユースケース**: 物語の時間軸表現
**入力**: 人物のクローズアップ映像

**プロンプト**:
```
この動画の被写体を、現在の年齢から約20歳若返らせてください（本人同意済み素材を前提）。
- 顔の輪郭・骨格は保持
- 肌のシワや毛穴を減らし、ハリと弾力を増す
- 髪の量を増やし、白髪を減らす
- 表情と発話の動きは完全に保持
- 服装と背景は変更しない
- フレーム間で安定
```

**コツ**: 老化の場合は「シワを増やし、髪を細く・薄く」を指定。
**ネガティブ**: 別人化、骨格の変形、表情の不自然さ

---

## 7. メイク変更（ナチュラル → 華やか）

**ユースケース**: ビフォーアフター動画、メイクチュートリアル
**入力**: ナチュラルメイクの人物クローズアップ

**プロンプト**:
```
この動画の被写体のメイクを、ナチュラルメイクから華やかなパーティーメイクに変更してください。
- 顔・髪型・表情は完全に保持
- アイメイク: ブラウンスモーキー、アイライナーを少し跳ね上げる
- リップ: 赤系のマット
- 頬: 控えめなチークとハイライト
- ライティングは元のシーンを保持
- フレーム間で安定
- 肌の質感は自然なまま
```

**コツ**: パーツごとに具体的に色や形状を指定。
**ネガティブ**: 顔のパーツ移動、肌のCG感、ちらつき

---

## 8. アクセサリー追加（眼鏡・帽子）

**ユースケース**: スタイル変更、季節感演出
**入力**: 人物の上半身映像

**プロンプト**:
```
この動画の被写体に、黒縁のセルフレーム眼鏡を追加してください。
- 眼鏡の位置・サイズ・パースは顔に正確にフィット
- 動作や顔の向きに追従
- レンズには周囲の光が控えめに反射
- 鼻と耳に自然に乗る（ずれ・浮きなし）
- 鼻の影が頬に投影
- フレーム間で安定
- 顔・髪型・服装は変更しない
```

**コツ**: 「鼻と耳に自然に乗る」「鼻の影」が違和感解消の鍵。
**ネガティブ**: 眼鏡の浮き、ずれ、頭の動きへの追従失敗

---

## 9. ジェスチャー・ポーズ追加

**ユースケース**: 棒立ち素材に動きを後付け
**入力**: 棒立ちで話している人物映像

**プロンプト**:
```
この動画の被写体に、話の内容に合わせた自然な手のジェスチャーを追加してください。
- 顔・表情・口の動き・声は完全に保持
- 両手を体の前で適度に動かす（強調・指差し・開く動作）
- ジェスチャーの頻度は2〜3秒に1回程度
- 動きは滑らかで過剰でない
- 衣服の袖の動きも整合的に追従
- 背景は変更しない
```

**コツ**: 頻度を「2〜3秒に1回」で指定すると過剰な動きを防げる。
**ネガティブ**: 過剰なオーバーアクション、手の歪み、リズムの不自然さ

---

## 10. リップシンク（口元を別音声に同期）

**ユースケース**: 吹き替え、多言語展開
**入力**: 元の発話映像 + 新音声

**プロンプト**:
```
この動画の被写体の口の動きを、添付した新しい音声に同期するように修正してください。
- 顔・表情の感情・視線・髪型・服装は完全に保持
- 口の形状を新音声の音素（子音・母音）に正確に対応
- 顎の動き・舌の動きも整合
- フレーム間で口元が安定（揺れ・歪みなし）
- 元のライティングと顔の質感を保持
- 喉や首の動きも発話に応じて自然に
```

**コツ**: 「音素に正確に対応」と書くと音節レベルでのシンクが向上する。
**ネガティブ**: 口の歪み、表情の崩れ、無関係な顔の変化
