# 02. シーン拡張・延長 (Frame Extension / Outpainting)

既存の動画フレームを時間軸または空間軸に拡張する。フレームを足す、尺を延ばす、見切れた被写体を補完するなど。

---

## 1. 9:16縦動画 → 16:9横動画にアウトペイント

**ユースケース**: TikTok素材をYouTube横動画として再利用
**入力**: 9:16 縦動画（5〜15秒）

**プロンプト**:
```
この9:16の縦動画を16:9の横動画にアウトペイントしてください。元のフレーム内容は中央に完全に保持してください。
- 左右に追加されるエリアは、既存の背景と連続する自然な拡張
- 被写体の動きやカメラパンに応じて拡張部分も整合性のあるモーション
- ライティングと色温度は元のフレームと一致
- 拡張部分にハッキリした境界線が出ないようシームレス処理
- 元のフレームレート・尺・音声はそのまま維持
```

**コツ**: 「中央に完全保持」を入れないと、リサイズで歪むことがある。
**ネガティブ**: 元フレームの歪み、左右に黒帯、拡張部分の不自然な切れ目

---

## 2. 短いクリップを2倍の尺に延長

**ユースケース**: 5秒クリップを10秒に伸ばしてSNS最適化
**入力**: 5秒のショートクリップ

**プロンプト**:
```
この5秒の動画を、自然な動きの補間で10秒に延長してください。
- 元の動きの始点から終点までを2倍の時間に滑らかに引き伸ばす
- 中間フレームをAI補間で生成（モーションブラーは抑制）
- 被写体の表情や動作の連続性を維持
- 音声がある場合はピッチ保持で2倍に引き伸ばす
- 背景の動きも整合的に延長
```

**コツ**: 単なるスローモーションと違い、動きの自然さを優先したい場合に有効。
**ネガティブ**: 静止フレームの繰り返し、ループ感、不自然なモーションブラー

---

## 3. シーンの前30秒を生成（前方延長）

**ユースケース**: クライマックスシーンに導入を後付け
**入力**: 既存のクライマックスシーン（10秒）

**プロンプト**:
```
この動画の冒頭に、自然につながる前段30秒を生成してください。
- 元のシーンの直前の状況を物語的に補完
- ロケーション、登場人物、ライティングは元のシーンと一致
- 元のシーンの最初のフレームに向かって動きが収束する
- カメラワークも元のシーンに自然に接続
- 音声は環境音ベースで違和感のないように
```

**コツ**: 「動きが収束する」と指定すると繋ぎ目が滑らかになる。
**ネガティブ**: 別人物の登場、ロケーション変更、時間帯の不一致

---

## 4. シーンの後30秒を生成（後方延長）

**ユースケース**: 冒頭シーンに続きを後付け
**入力**: 冒頭シーン（10秒）

**プロンプト**:
```
この動画の末尾に、自然につながる後続30秒を生成してください。
- 元の最終フレームから動きが連続して始まる
- 物語的に自然な展開（人物の次の行動、シーン展開）
- ライティングと色調は元のシーンを継承
- カメラワークも違和感なく続く
- 環境音や効果音も連続性を持って生成
```

**コツ**: 「動きが連続して始まる」を入れると瞬間移動を防げる。
**ネガティブ**: 唐突なシーン転換、新しい人物の突然の登場、時間飛び

---

## 5. 切れた被写体の体の一部を補完

**ユースケース**: 構図ミスで切れた手足を後から補完
**入力**: 被写体の足や手が見切れている動画

**プロンプト**:
```
この動画で見切れている被写体の足元（腰から下）を、自然に補完してください。
- 既存の腰から上の動きと一貫性のある足の動きを生成
- 衣服のスタイル・素材・色は既存部分と連続
- 床面や地面の質感を背景に応じて整合的に追加
- ライティングと影は既存部分と一致
- カメラの動きにも追従
```

**コツ**: 「衣服のスタイル・素材・色」を明示すると違和感が減る。
**ネガティブ**: 切れた境界の不自然なつなぎ目、衣服スタイルの不一致

---

## 6. シーン手前の風景を奥行き方向に拡張

**ユースケース**: 平面的なショットに奥行きを後付け
**入力**: フラットな構図の風景動画

**プロンプト**:
```
この動画の構図に、カメラ手前側に奥行きを拡張してください。
- 元の構図はミドル〜ロング距離として再配置
- 手前にフォアグラウンド要素を追加（草、枝、岩などロケーションに合うもの）
- 被写界深度を浅くしてフォアグラウンドを若干ボケさせる
- カメラの動きにフォアグラウンドも追従
- 既存のライティングと整合
```

**コツ**: フォアグラウンドのボケが奥行き感を演出する。
**ネガティブ**: ハッキリしすぎたフォアグラウンド、構図の遠近矛盾

---

## 7. ループ動画化（先頭と末尾を滑らかに繋ぐ）

**ユースケース**: SNSのループ動画、ライブ壁紙
**入力**: 短い動画クリップ（2〜10秒）

**プロンプト**:
```
この動画をシームレスにループする動画に再構成してください。
- 末尾フレームから先頭フレームへ滑らかに遷移する補間フレームを末尾に追加
- 動きの流れが自然にループ点で折り返す
- 音声がある場合は末尾と冒頭をクロスフェード
- 元の動きの大筋は完全に保持
- ループ点が視聴者に気づかれない自然さ
```

**コツ**: 「ループ点が気づかれない」と書くと境界処理が丁寧になる。
**ネガティブ**: 急な巻き戻し感、はっきりした繋ぎ目

---

## 8. 切れた音声に合わせて映像尺を延長

**ユースケース**: 音声尺＞映像尺の編集差分解消
**入力**: 短い映像（10秒）+ 長い音声（15秒）

**プロンプト**:
```
この映像を、添付の15秒音声と完全に同期するように尺延長してください。
- 映像末尾に5秒の自然な続きを生成
- 元の被写体・ロケーション・カメラワークの連続性を維持
- 音声のリズム・テンポに合わせた動きを追加部分に組み込む
- ライティングと色は元と一致
- 全体で15秒・音声同期
```

**コツ**: 音声のリズムに合わせた動き生成は短尺PV制作で有効。
**ネガティブ**: 映像の急停止、音ズレ、被写体の唐突な消失

---

## 9. 同シーンの別アングル映像を生成

**ユースケース**: 1カットしかない素材から複数アングルを生成
**入力**: 1アングルの動画

**プロンプト**:
```
このシーンの別アングル（被写体の右側45度から）の映像を新規生成してください。
- 元シーンと完全に同じタイミング・尺・動き
- ロケーション、被写体、衣服、ライティングは完全一致
- カメラのみ右側45度に配置
- 同じレンズ画角（35mm相当）
- 音声は元のシーンと同期
```

**コツ**: アングルを「右側45度」のように数値で指定。
**ネガティブ**: 別シーンの生成、時間軸のズレ、被写体の見た目の不一致

---

## 10. ハイライトのみ抽出して短尺化（逆拡張）

**ユースケース**: 60秒素材からSNS用15秒抽出
**入力**: 60秒の動画

**プロンプト**:
```
この60秒の動画から、最も視覚的・物語的にインパクトのある15秒を抽出してください。
- ハイライト判定基準: 動きの大きい瞬間、表情変化、被写体の動的アクション
- 抽出した15秒は自然なジャンプカットで繋ぐ
- カット間のテンポはリズミカル（平均1.5秒/カット）
- 音声は元のセリフ・効果音の重要部分を優先
- 冒頭3秒に最強の引きを配置
```

**コツ**: 「冒頭3秒に最強の引き」がSNSでは決定的。
**ネガティブ**: 単調な時系列カット、退屈な間、冗長なシーン
